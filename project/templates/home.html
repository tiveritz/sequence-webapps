The project Sequences consists of a collection of web applications that allow to create, manage, edit and view Step-by-Step instructions.


An important key aspect ist the modularity of the documentation. Reusable steps, explanations, pictures, links and so on are a core concept of the technology behind it.

Viewer

The Viewer is a web application to consume the content. View comprehensive Sequence Step-by-Step Instructions and get things done quicker.

What do we need documentation for?


Not everyone is positively inclined towards documentation, especially when it comes to Software development. It is a lot of work and generally speaking those things are just lies waiting to be read, if not properly maintained.


Do never forget, that code should be self explanatory and not much documentation should be neccessary.


However there is a lot of stuff as well, especially regarding development and deployment procedures which evolved over time and aren't in any way self explanatory. And those are the things that need proper documentation.


Don't spend so much time explaining new staff how things are done. Starting creating Step-by-Step procedures.
